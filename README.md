# בקשה להארכת גרייס - מזרחי טפחות

שום מידע (ת.ז / מספר חשבון / חתימה) לא נשלח לשום שרת. הלקוח שולח את הקובץ בעצמו לבנק: `grace@umtb.co.il`.
